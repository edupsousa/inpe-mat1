INPE / CAP / Matemática Computacional I

Aqui você encontrará implementações na linguagem JavaScript para os tópicos desenvolvidos durante a disciplina Matemática Computacional I, do curso de Computação Aplicada, do Instituto Nacional de Pesquisas Espaciais.

Utilize os links do menu de navegação para acessar os exemplos das funcionalidades desenvolvidas. Atualmente está disponível:

  • Interpolação Polinomial pelo Método de Eliminação Gaussiana
  • Integração Numérica pela Regra do Ponto Médio (Retângulo)
  • Integração Numérica pela Regra do Trapézio
  • Problema de Valor Inicial / Equação Diferencial Ordinária
    • Método de Euler
    • Runge Kutta de 4ª Ordem
